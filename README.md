🚀 Prit Desai - GitHub Portfolio


Welcome to my GitHub portfolio website! This project is built using GitProfile and customized to showcase my projects, skills, experiences, and certifications.

🔥 Live Demo
🌎 Visit My Portfolio

📌 Features
✅ Showcases my GitHub repositories dynamically.
✅ Displays skills, experiences, and certifications.
✅ Fully responsive and built with React.js, TailwindCSS, DaisyUI.
✅ Integrated daisyUI themes with light/dark mode toggle.
✅ Deployed using GitHub Pages.

🛠️ Technologies Used
Frontend: React.js, Vite.js, TailwindCSS, DaisyUI
Backend: GitHub API
Deployment: GitHub Pages
