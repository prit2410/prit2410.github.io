## 🛠️ How to Run Locally

You can view and test this project on your local machine using either of the following methods:

### Method 1: Direct File Open (Easiest)
1. Clone or download this repository to your computer.
2. Navigate to the project folder.
3. Double-click `index.html` (or right-click -> Open With -> Chrome/Edge/Firefox) to view it in your browser.

### Method 2: Live Server (Recommended for Editing)
1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension from the VS Code marketplace.
3. Right-click on `index.html` in the file explorer and select **"Open with Live Server"**.
4. The project will open in your browser and automatically refresh whenever you save changes.

## 🌐 Deployment

This project is purely frontend (static files) and is optimized for easy deployment on modern hosting platforms.

**To deploy with Vercel:**
1. Push this repository to your own GitHub account.
2. Log in to Vercel and click **Add New Project**.
3. Import the repository from your GitHub.
4. Click **Deploy**. Vercel will automatically generate a live, shareable URL for your portfolio.
